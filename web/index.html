Everything you need to navigate independently

Avryd handles every app, screen, and situation on Windows.

🎙

Natural Speech

High-quality Piper TTS voices with adjustable rate, pitch, and volume. Supports multiple languages with offline capability.

⌨

Keyboard Navigation

Full keyboard control with customizable shortcuts. Navigate by headings, buttons, links, form fields, and landmarks.

🔍

UIA Accessibility Engine

Built on Windows UI Automation. Reads every accessible app including Office, Chrome, Edge, File Explorer, and modern Windows apps.

📷

OCR Fallback

For apps without accessibility support, Avryd uses OCR to read the screen and builds a virtual accessibility tree automatically.

🗣

Voice Commands

Control your computer hands-free. Say "Open Chrome", "Volume up", "Next item", or install custom command plugins.

⠿

Braille Support

Full Braille keyboard input with 6-dot and 8-dot modes. Navigate and type using any connected Braille device.

🔒

Privacy First

Your screen content never leaves your device unless you enable cloud features. Passwords are never spoken by default.

🧩

Plugin System

Extend Avryd with plugins for custom apps, languages, and commands. Install directly from the Avryd GitHub plugin repository.
